# ExceptionOverflow

Find exceptions on StackOverflow inside your terminal.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'exception_overflow'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exception_overflow
